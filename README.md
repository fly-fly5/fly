# Python 数据分析训练中心

专为商务数据分析专业打造的 Python 学习平台，涵盖基础知识、核心技能与十个实战训练项目。

## 功能板块

- **知识体系** — 6大 Python 基础知识模块（可展开查看详情）
- **核心技能** — NumPy、Pandas、Matplotlib、Seaborn、爬虫、机器学习
- **训练项目** — 10个从入门到进阶的实战项目（可展开查看学习目标）
- **学习路线** — 16周系统化学习计划 + Mermaid 流程图
- **技能雷达** — ECharts 可视化技能掌握目标
- **知识练习** — 10道选择题，答错显示详细解析

## 技术栈

- HTML5 + CSS3（淡紫梦幻主题）
- ECharts（雷达图）
- Mermaid（流程图）
- 原生 JavaScript（交互逻辑）
